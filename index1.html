About Me

Tentang Saya

Saya percaya bahwa teknologi terbaik adalah yang tidak terasa seperti teknologi — melainkan seperti sihir yang bekerja diam-diam di balik layar.

Dengan lebih dari 3 tahun pengalaman di dunia pengembangan web dan sistem monitoring IoT, saya membangun solusi digital — dari platform web enterprise hingga perangkat embedded yang bekerja real-time 24/7 di lingkungan infrastruktur telekomunikasi nasional.

Saya bukan hanya seorang developer — saya seorang problem solver. Sistem monitoring berbasis ESP32 untuk suhu, getaran/gempa, BBM genset, power panel PKG, AC server — setiap sensor, setiap data point harus akurat dan bisa dipercaya.

Saat tidak coding, saya menulis artikel tentang teknologi, filosofi hidup, dan lingkungan sekitar — karena developer terbaik adalah mereka yang memahami dunia lebih dari sekadar layar komputer.
